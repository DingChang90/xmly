# 喜马拉雅极速版

1.fork此项目 </br>
2. [注册喜马拉雅极速版账号](https://m.ximalaya.com/xmds-node-spa/apps/speed-growth-activities/web-earn/landing?parentUserId=71848495&channel=1&srcChannel=weixin&srcPicUrl=https%3A%2F%2Ffdfs.xmcdn.com%2Fgroup63%2FM06%2F26%2F3C%2FwKgMcl0Z7-bgXtm3AAAP_-xdbjM707.png&srcText=2) </br>
3.下载喜马拉雅极速版，必须指定低版本APP，仅提供安卓版，[点击下载](http://file.pgdtcz.com/#s/6hWbX2Zg) </br>
4.安卓安装HttpCanary工具，或者安卓指定代理MAC安装Charlse软件，抓取请求cookie。</br>
5.在自己fork的仓库里，新建Secrets，NAME：XMLY_SPEED_COOKIE，Value：抓取的cookie值。</br>
6.新建Secrets，NAME：SCKEY，Value：Server酱推送的SCKEY
